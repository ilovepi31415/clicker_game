# Make a Million: A Clicker Game

### Summary
Have you ever wanted to make a Million Dollars? Well, now you can (in this game). Click your way there, earning upgrades and more as you go. Your progress will be saved if you exit the game, so you can come back stronger the next time. Even once you're done, keep trying to improve your time and be the fastest millionaire in the universe.

### Requirements
This program was created in Pygame, so it and Python are required to run it.

### Usage
Run the index.py file, and the game will start. Click your way to a Million Dollars, after which you will see the time it took you to reach it. Progress will reset if `savefile.json` is deleted or the clear progress X button is clicked in game.
